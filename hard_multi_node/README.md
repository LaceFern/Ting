## How to Build a Vivado Project Using Chisel Framework and HLS Modules

Step 1. See ./hls/README.md to generate HLS IP and related Chisel wrapper in the specified Vivado project

Step 2. See ./chisel/README.md to analyze all top-level Chisel wrappers for HLS modules based on the HLS IP wrappers generated in ./hls/README.md in the specified directory

Step 3. See ./chisel/README.md to complete the final Chisel top file using the top-level Chisel wrapper, and generate .sv file in the specified Vivado project
